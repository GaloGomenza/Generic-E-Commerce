¿Que es Generic E-Commerce?

🛍️

Backend de una aplicacion web de e-commerce

API REST que expone toda la funcionalidad de un sistema de comercio electronico: gestion de usuarios, catalogo de productos, carrito de compras, procesamiento de ordenes y sistema de calificaciones.

Consumible desde cualquier frontend (Angular, React, Vue) o herramienta como Postman.

Funcionalidades principales

  • Registro y autenticacion de usuarios
  • Roles: ADMIN (gestiona catalogo) y USER (compra)
  • CRUD completo de categorias y productos
  • Carrito de compras persistente por usuario
  • Creacion y cancelacion de ordenes
  • Sistema de calificaciones y reviews
  • Filtros y busquedas en cada modulo

Organizacion por modulos

Cada modulo de negocio (category, product, cart, order, review, user) sigue la misma estructura de paquetes:

  • controller/ — Endpoints REST
  • service/ — Interface + implementacion
  • repository/ — Acceso a datos JPA
  • model/ — Entidades JPA
  • dto/request/ — DTOs de entrada
  • dto/response/ — DTOs de salida

📁 config/ — Configuracion centralizada (seguridad, CORS, H2, datos iniciales)

📁 shared/ — Componentes transversales (excepciones, DTOs genericos, seguridad)
